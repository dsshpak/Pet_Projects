# Pet_Project_AB
Анализ результатов A/B-теста
Выполнил подсчет результатов A/B-теста: проверка корректности проведения теста и целостнотси данных, подсчет основных метрик и поиск статистически значимых различий между группами. Сделал выводы о влиянии изменений на поведение пользователей и подготовил рекомендации по внедрению решения.
