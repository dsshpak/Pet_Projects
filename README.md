Аналитические проекты, выполненные для практики работы с данными и продуктовыми метриками

1. Дашборд по аварийному жилью
Разработал аналитический дашборд для мониторинга состояния аварийного жилищного фонда.
Провел первичную обработку данных, выделил ключевые показатели (количество объектов, динамика признания жилья аварийным, распределение по регионам), реализовал визуализацию в Yandex Datalens для удобства принятия управленческих решений службой ЖКХ

2. Анализ результатов A/B-теста
Выполнил подсчет результатов A/B-теста: проверка корректности проведения теста и целостнотси данных, подсчет основных метрик и поиск статистически значимых различий между группами. Сделал выводы о влиянии изменений на поведение пользователей и подготовил рекомендации по внедрению решения
